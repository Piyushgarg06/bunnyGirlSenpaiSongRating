# 🎵 Bunny Girl Senpai Song Ranking Website

Welcome to my **Bunny Girl Senpai Song Ranking Website**!

This is a personal web project where I share **my own ranking** of the same song sung in the voices of different characters from the anime **Seishun Buta Yarou wa Bunny Girl Senpai no Yume wo Minai** (Bunny Girl Senpai).  

I built this site as a fun way to practice my **HTML, CSS, and JavaScript** skills—and because I’m a huge fan of Bunny Girl Senpai!

---

## ✨ Features

- 🎶 **Same song, different voices**  
  Listen to how the same track sounds when sung by various characters.

- ⭐ **My personal ranking**  
  The site showcases how **I** ranked the different versions based on how much I liked each character’s singing voice.

- 🎨 **Custom design**  
  Clean and stylish UI designed entirely by me.

- 💻 **Responsive layout**  
  Looks great on both desktop and mobile devices.
